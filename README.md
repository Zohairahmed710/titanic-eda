# Titanic Survival – Exploratory Data Analysis
Main goals:
1.To check my skillset.
2.To analyse what are the factors affecting survival rate.
3.To show non-technical person what is the importance of EDA.

There are several tools used in this project some of them are:
1.Google colab - virtual environment
2.Pandas - data handling 
3.Matplotlib - ploting graphs
4.Seaborn - advance stats

What I've found from this EDA?
This is a good question, and I'm really intrested to discuss this in this README file, here is some insights:
1.Among all people like childrens, man, and woman. Woman survived most.
2.In first-class people paid high-prices and analysis clearly showed that first-class passengers survival rate is higher.
3.One small and interesting fact is that Cherbourg towns males have high survival rate compared to other towns.
