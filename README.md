# Titanic Survival – Exploratory Data Analysis
Main goals:
1. To check my skillset.
2. To analyse what are the factors affecting survival rate.
3. To show a non-technical person what is the importance of EDA.

There are several tools used in this project, some of them are:
1. Google Colab – virtual environment
2. Pandas – data handling 
3. Matplotlib – plotting graphs
4. Seaborn – advanced statistical plots

What I've found from this EDA?
This is a good question, and I'm really interested to discuss this in this README file. Here are some insights:
1. Among all groups – children, men, and women – women had the highest survival rate.
2. In first-class, people paid higher fares, and the analysis clearly showed that first‑class passengers had a higher survival rate.
3. One small and interesting fact is that males from Cherbourg town had a high survival rate compared to other towns.
